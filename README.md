# ArtPulse  
### Цифровая галерея современного искусства  

Веб-приложение, разработанное в рамках курсовой работы по дисциплине *«Веб-проектирование»* в Нижегородском государственном архитектурно-строительном университете (ННГАСУ).  
Объединяет принципы современной frontend-разработки и концептуального дизайна: полностью клиентская реализация цифровой платформы для демонстрации генеративного, видео- и нейро-искусства.

---

## Общее описание

**ArtPulse** — многостраничное адаптивное веб-приложение, имитирующее работу цифровой галереи. Проект обеспечивает целостный пользовательский опыт на десктопных и мобильных устройствах: от семантической структуры и плавных анимаций до персонализации контента и локального хранения данных.

Реализация соответствует и превосходит все требования к курсовой работе:
- 3 функциональные страницы (Главная, Художник, Посетить);
- Блочная, семантически корректная вёрстка с вынесенными стилями;
- Полная адаптивность (медиа-запросы, touch-интерфейс, мобильное меню);
- 6 независимых JavaScript-модулей (авторизация, избранное, навигация, карусель, валидация форм, main);
- Сохранение состояния между сессиями через `localStorage`;
- Блок «О разработчике» с персональным фото.

Без использования backend-инфраструктуры и фреймворков — только HTML, CSS, JavaScript.

## Ключевые возможности

| Возможность | Реализация | Технологии |
|-------------|------------|------------|
| **Система авторизации** | Вход / регистрация с сохранением сессии | `localStorage`, валидация по email и длине пароля, модальное окно |
| **Избранное** | Добавление/удаление работ; данные сохраняются при перезагрузке | Делегирование событий, `data-*` атрибуты, JSON-сериализация |
| **Видео-карусель** | Автопрокрутка, ручное управление, адаптивная высота | `<video autoplay muted loop playsinline>`, JS-таймеры, CSS-трансформации |
| **Адаптивное меню** | «Бургер»-меню с плавным раскрытием и закрытием по клику вне области | `classList.toggle()`, `click` вне элемента |
| **Контактная форма** | Проверка заполнения и валидация email | Регулярные выражения, динамические уведомления |
| **Информация о разработчике** | Блок с аватаром, ФИО, ролью и контактами | Flexbox, семантическая разметка, локальное фото |

## Техническая реализация

- **Разметка**: HTML5 с семантическими тегами (`<header>`, `<section>`, `<article>`, `<footer>`);
- **Стилизация**: Единый CSS-файл (`style.css`), без inline-стилей. Использованы:
  - CSS-переменные для управления цветовой палитрой;
  - Flexbox и Grid для компоновки;
  - `backdrop-filter: blur(...)` для эффекта «стекла» в навигации;
  - плавные переходы (`transition: all 0.3s ease`);
  - медиа-запросы (breakpoint: 768px);
- **Логика**: Чистый JavaScript (ES6+):
  - Модульная структура (разделение по файлам/задачам);
  - Работа с DOM через `querySelector`, `addEventListener`;
  - Сохранение состояния в `localStorage` с JSON-сериализацией;
  - Безопасная инициализация через `DOMContentLoaded`.

## Запуск

Проект не требует сборки и сервера.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/alexhet/ArtPulse_site.git
2. Откройте любой HTML-файл в браузере:
   index.html — главная страница;
   artist.html — страница художника;
   visit.html — информация о посещении и контакты.
Рекомендуется использовать расширение Live Server в WebStorm или VS Code для корректной загрузки локальных изображений и видео.
